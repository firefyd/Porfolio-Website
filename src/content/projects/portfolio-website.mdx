## Tech Stack

- Version Control

## Architecture Design
- Folder Structure
src/ with components layouts pages content and styles

- Content management
    Astro content collections with schemas for projects and dynamic routing

- Core components
    Header, footer, baselayout, themetoggle, and secondary

- Pages
    Home, Projects, about, content


## Design 

- Light dark mode
    Robust variables to ensure ease of switching, auto detection of user preferences

- Mobile width
    Burger menu

- Accesibility (WCAG)
    Used WCAG friendly colour and contrast levels
